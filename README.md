Device configuration for the Wiko Darkmoon (MT6582)
===============================

Android 4.2.2 - cm-10.1

-------------------------------

To build first get the files from working phone :

$cd device/wiko/darkmoon

$./extract-files.sh

Build :

$. build/envsetup.sh

$brunch cm_darkmoon-eng
